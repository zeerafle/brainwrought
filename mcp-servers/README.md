# MCP Servers

This directory contains external MCP servers used by the project.

## Setup

### TikTok MCP
```bash
cd mcp-servers
git clone https://github.com/Seym0n/tiktok-mcp.git
cd tiktok-mcp
npm install
npm run build
```

Required environment variable:
- `TIKNEURON_MCP_API_KEY`: Get from https://tikneuron.com/tools/tiktok-mcp
